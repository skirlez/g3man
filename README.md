# Forgery Manager
A mod manager for the https://github.com/Skirlez/nubbys-forgery/ modloader

## Compiling
### Nix/NixOS
Clone the repository and run `nix-shell` inside of it. Then run `make`. Output is in `out/forgerymanager`. You may need nix-ld to run it.
